# Forex API Server\n\nReal-time forex data API server deployed on Vercel.\n\n## Endpoints\n\n- `GET /` - Main page with trading interface\n- `GET /admin` - Admin panel for API key management\n- `POST /api/generate-key` - Generate new API key\n- `GET /api/keys` - List all API keys\n- `GET /ws` - WebSocket endpoint info\n\n## Usage\n\n1. Visit the main page to see the trading interface\n2. Generate API keys from `/admin` page\n3. Use WebSocket connection for real-time data\n\n## API Keys\n\nDefault keys:\n- `demo-key-123`\n- `test-key-456`\n- `live-key-789`\n\n## Deployment\n\nDeployed on Vercel with serverless functions.\n\n```bash\nnpm run deploy\n```\n
